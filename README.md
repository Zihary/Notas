#  Notas Unison

Aplicación desarrollada en Flutter para la gestión de notas con almacenamiento local usando Hive.

---

##  Descripción

Notas Unison es una aplicación móvil y web que permite crear, editar y eliminar notas de manera sencilla e intuitiva.

La aplicación ofrece una interfaz moderna tipo Grid, selección de colores personalizados para cada nota y almacenamiento local persistente utilizando Hive.

Está diseñada para organizar ideas, tareas o apuntes de forma visual y atractiva.

---

##  Funcionalidades

-  Crear notas
-  Editar notas
-  Eliminar notas
-  Selección de colores personalizados
-  Diseño tipo Grid
-  Almacenamiento local con Hive
-  Interfaz moderna con degradado y animaciones

---

##  Tecnologías Utilizadas

- **Flutter 3.x** – Framework para desarrollo multiplataforma
- **Dart 3.x** – Lenguaje de programación
- **Hive ^2.2.3** – Base de datos local ligera
- **hive_flutter ^1.1.0** – Integración de Hive con Flutter
- **Material Design** – Sistema de diseño visual
